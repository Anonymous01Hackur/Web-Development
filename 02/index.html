Frontend development is the development of the graphical user interface of a website, throught the use of HTML, CSS, JAVASCRIPT, so that the user can view and interact withe that website.



  • HTML: The Hypertext language or HTML is the stander markup language fro document designed to be display in the we browser.



  • CSS: Cascading style sheets (CSS) is the style sheet language used to describing the presentation of a document with html.




  • JAVASCRIPT: JavaScript is a high-level, dynamic, and interpreted programming language that is commonly used to create interactive and dynamic web pages. It is an essential component of modern web development, alongside HTML and CSS. JavaScript allows developers to add complex and rich features to websites, making them more engaging and user-friendly.
